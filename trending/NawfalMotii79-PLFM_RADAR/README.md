# NawfalMotii79/PLFM_RADAR

Open-source, low-cost 10.5 GHz PLFM phased array RADAR system

项目定位  
一个开源、低成本的 10.5 GHz 相控阵雷达系统，用于实现高精度目标探测与跟踪。

核心功能  
- 实现 10.5 GHz 频段的雷达信号发射与接收  
- 支持相控阵天线波束控制  
- 提供目标检测与距离测量功能  
- 开源设计便于二次开发与硬件扩展  

技术特点  
采用 PLSQL 编程语言实现雷达控制逻辑，结合低成本硬件构建高性能相控阵雷达系统，具备良好的可扩展性与研究价值。

适用场景  
- 雷达系统研究与教学  
- 低预算的科研项目  
- 电子工程与通信领域的实验平台  
- 业余无线电爱好者项目  
- 智能交通与安防监控应用  

推荐理由  
该项目为开发者提供了一个低成本、高灵活性的雷达系统参考，适合用于学习、研究和实际应用，具有较高的实践价值和技术深度。
