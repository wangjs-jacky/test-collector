# tirth8205/code-review-graph

Local knowledge graph for Claude Code. Builds a persistent map of your codebase so Claude reads only what matters — 6.8× fewer tokens on reviews and up to 49× on daily coding tasks.

1. **项目定位**  
该项目是一个用于 Claude Code 的本地知识图谱工具，旨在通过构建代码库的持久化映射，减少代码审查和日常编码任务中的 token 使用量。

2. **核心功能**  
- 构建代码库的持久化知识图谱  
- 提升 Claude Code 的代码理解效率  
- 减少代码审查时的 token 消耗  
- 支持增量更新和静态分析  
- 优化 LLM 在代码任务中的表现

3. **技术特点**  
使用 Python 实现，结合 Tree-sitter 进行静态分析，构建结构化的知识图谱，提升 AI 对代码的理解和处理效率。

4. **适用场景**  
- 代码审查流程优化  
- 日常开发中减少 LLM 调用成本  
- 需要高效处理大型代码库的场景  
- 与 Claude Code 集成的开发环境  
- 需要知识图谱支持的 AI 编程工具

5. **推荐理由**  
该工具有效提升 AI 编程效率，降低 token 消耗，适合需要高频使用 AI 辅助开发的开发者和团队。
