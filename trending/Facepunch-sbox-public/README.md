# Facepunch/sbox-public

s&box is a modern game engine, built on Valve's Source 2 and the latest .NET technology, it provides a modern intuitive editor for creating games

项目定位  
s&box 是一个基于 Valve Source 2 和 .NET 的现代游戏引擎，提供直观的编辑器用于游戏开发，旨在简化游戏制作流程。

核心功能  
- 基于 Source 2 引擎的高性能渲染与物理模拟  
- 提供可视化编辑器用于游戏场景和逻辑构建  
- 支持 C# 脚本开发，提升开发效率  
- 集成现代化工具链，便于团队协作  

技术特点  
s&box 结合了 Source 2 的强大性能与 .NET 的现代开发体验，采用 C# 语言实现，具备良好的可扩展性和跨平台潜力。

适用场景  
- 独立游戏开发者快速构建原型  
- 教育机构用于游戏开发教学  
- 小型团队进行 3D 游戏项目开发  
- 开发者探索 Source 2 引擎的新特性  

推荐理由  
s&box 结合了经典引擎与现代开发技术，适合希望在 Source 2 上进行创新开发的开发者，提供高效、易用的游戏开发环境。
