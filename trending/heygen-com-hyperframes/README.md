# heygen-com/hyperframes

Write HTML. Render video. Built for agents.

项目定位  
Hyperframes 是一个用于生成视频的框架，允许开发者通过 HTML 编写内容并高效渲染为视频，专为 AI 代理设计。

核心功能  
- 使用 HTML 构建动画和视频内容  
- 支持与 AI 代理集成  
- 提供高效的视频渲染流程  
- 集成 GSAP 和 Puppeteer 实现复杂动画  
- 支持多种视频格式输出

技术特点  
基于 TypeScript 构建，结合 HTML、CSS 和动画库实现高效渲染，利用 Puppeteer 和 FFmpeg 进行自动化视频生成，适合 AI 相关应用开发。

适用场景  
- AI 代理的可视化输出  
- 动画视频自动生成  
- 交互式网页内容转视频  
- 教育或营销视频制作  
- 快速原型开发与测试

推荐理由  
Hyperframes 为 AI 代理提供强大的视频生成能力，简化了从 HTML 到视频的转换流程，适合需要自动化视频生产的开发场景。
