# google-labs-code/design.md

A format specification for describing a visual identity to coding agents. DESIGN.md gives agents a persistent, structured understanding of a design system.

项目定位  
一个用于描述视觉身份的格式规范，帮助编码代理理解设计系统，实现设计与代码的统一。

核心功能  
- 定义设计系统的结构化描述  
- 提供持久化的视觉身份信息  
- 促进设计与开发的协作  
- 支持自动化设计转换  

技术特点  
基于 TypeScript 构建，强调类型安全和结构化数据，便于集成到开发流程中，提升设计一致性。

适用场景  
- 设计系统构建与维护  
- 自动化 UI 生成  
- 跨团队设计与开发协作  
- AI 编码代理训练  

推荐理由  
为设计与代码的无缝对接提供标准化方案，适合需要高效设计系统管理的团队和开发者。
