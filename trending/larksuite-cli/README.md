# larksuite/cli

The official Lark/Feishu CLI tool, maintained by the larksuite team — built for humans and AI Agents. Covers core business domains including Messenger, Docs, Base, Sheets, Calendar, Mail, Tasks, Meetings, and more, with 200+ commands and 19 AI Agent Skills.

项目定位  
Lark/Feishu 的官方命令行工具，用于高效管理企业协作功能，提升开发者和用户操作效率。

核心功能  
- 提供 200+ 命令覆盖 Messenger、Docs、Base 等核心业务模块  
- 支持 AI Agent 技能集成，增强自动化能力  
- 实现与 Lark/Feishu 平台深度集成，简化日常操作  

技术特点  
采用 Go 语言开发，具备高性能和跨平台特性，结构清晰，便于维护和扩展，适合企业级应用。

适用场景  
- 开发者快速调用 Lark/Feishu API  
- 企业用户通过 CLI 管理团队协作任务  
- 自动化办公流程，提升工作效率  

推荐理由  
功能全面、性能稳定，适合需要深度集成 Lark/Feishu 的开发者和企业用户，是提升协作效率的利器。
