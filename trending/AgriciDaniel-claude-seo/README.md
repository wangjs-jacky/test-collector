# AgriciDaniel/claude-seo

Universal SEO skill for Claude Code. 19 sub-skills, 12 subagents, 3 extensions (DataForSEO, Firecrawl, Banana). Technical SEO, E-E-A-T, schema, GEO/AEO, backlinks, local SEO, maps intelligence, Google APIs, and PDF/Excel reporting.

项目定位  
一个为 Claude Code 设计的通用 SEO 工具，提供多种子技能和扩展功能，帮助优化网站内容和提升搜索引擎排名。

核心功能  
- 技术 SEO 分析  
- E-E-A-T 评估与优化  
- 地理/区域 SEO 支持  
- 网站爬取与数据抓取  
- 生成 PDF/Excel 报告  

技术特点  
基于 Python 开发，集成多个第三方 API（如 DataForSEO、Firecrawl），支持自动化 SEO 流程，具备模块化设计和可扩展性。

适用场景  
- 网站内容优化  
- 搜索引擎排名提升  
- 在线营销自动化  
- 多地区 SEO 管理  
- SEO 数据分析与报告  

推荐理由  
功能全面，适合需要自动化 SEO 的开发者和营销人员，结合 Claude Code 提升效率，开源且社区活跃。
