# safishamsi/graphify

AI coding assistant skill (Claude Code, Codex, OpenCode, Cursor, Gemini CLI, GitHub Copilot CLI, OpenClaw, Factory Droid, Trae, Google Antigravity). Turn any folder of code, docs, papers, images, or videos into a queryable knowledge graph

项目定位  
一个将代码、文档等资料转化为可查询知识图谱的 AI 编程助手工具，支持多种 AI 编码助手。

核心功能  
- 支持多种 AI 编码助手（Claude Code, Codex 等）  
- 将文件夹内容构建为知识图谱  
- 提供结构化数据查询能力  
- 支持代码、文档、图像等多种类型数据  

技术特点  
基于 Python 实现，结合多种 AI 模型和知识图谱技术，实现多源数据的智能整合与查询。

适用场景  
- 代码知识管理与检索  
- 文档与论文知识提取  
- 多媒体数据结构化处理  
- 开发者工具链集成  
- AI 助手技能扩展  

推荐理由  
功能强大且支持多种 AI 工具，适合需要高效管理和查询复杂数据的开发者和研究人员。
