# datawhalechina/easy-vibe

💻 vibe coding 2026 | Your first modern programming course for beginners to master step by step.

项目定位  
一个面向初学者的现代编程课程，旨在通过逐步引导帮助用户掌握编程技能，结合 AI 工具提升学习效率。

核心功能  
- 提供结构化编程教程  
- 集成 AI 编程助手（如 DeepSeek、GPT）  
- 支持低代码/无代码开发  
- 包含 VSCode 插件和工作流工具  
- 与多种 AI 模型兼容  

技术特点  
基于 JavaScript 构建，融合 AI 技术与现代开发工具，强调易用性和可扩展性，适合初学者快速上手。

适用场景  
- 编程入门学习者  
- 希望使用 AI 工具辅助编程的开发者  
- 寻找低代码开发方案的团队  
- 教育机构用于教学内容开发  
- 个人项目快速原型设计  

推荐理由  
适合编程新手系统学习，结合 AI 工具提升效率，内容结构清晰，涵盖现代编程所需技能，具备良好的扩展性和实用性。
