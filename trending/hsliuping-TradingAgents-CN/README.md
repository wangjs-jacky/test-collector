# hsliuping/TradingAgents-CN

基于多智能体LLM的中文金融交易框架 - TradingAgents中文增强版

**项目定位**  
基于多智能体LLM的中文金融交易框架，用于实现自动化交易策略，提升金融市场的智能化交易能力。

**核心功能**  
- 支持多智能体协同交易  
- 集成自然语言处理与金融数据分析  
- 提供回测与实时交易功能  
- 可扩展性强，支持自定义策略  
- 中文环境优化，适合国内金融市场

**技术特点**  
采用多智能体系统与大语言模型结合，具备良好的可扩展性和适应性，专注于金融交易场景的智能化升级。

**适用场景**  
- 量化交易策略开发与测试  
- 金融数据智能分析与预测  
- 自动化交易系统构建  
- 金融研究与教学实验  
- 中文市场交易策略优化

**推荐理由**  
该项目融合了多智能体与LLM技术，为中文金融交易提供创新解决方案，适合开发者和研究者快速构建智能交易系统。
