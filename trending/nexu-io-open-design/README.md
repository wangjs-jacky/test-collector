# nexu-io/open-design

🎨 Local-first, open-source alternative to Anthropic's Claude Design. ⚡ 19 Skills · ✨ 71 brand-grade Design Systems · 🖼️ sandboxed preview · 📦 HTML/PDF/PPTX export. 🤖 Runs on Claude Code / Codex / Cursor / Gemini CLI / OpenCode / Qwen / Copilot / Hermes / Kimi CLI.

**项目定位**  
一个本地优先的开源替代方案，用于 Anthropic 的 Claude Design，提供设计系统和生成式 AI 工具支持。

**核心功能**  
- 提供 71 个品牌级设计系统  
- 支持 HTML/PDF/PPTX 导出  
- 沙盒预览功能  
- 集成多种 AI 编程工具如 Claude Code、Codex 等  
- 具备 19 种技能模块  

**技术特点**  
使用 TypeScript 构建，强调本地运行与安全性，结合 AI 工具实现高效设计流程，具备良好的可扩展性。

**适用场景**  
- 设计师快速构建品牌设计系统  
- 开发者集成 AI 工具进行自动化设计  
- 企业需要本地化部署的 AI 设计平台  
- 教育机构用于 AI 设计教学  
- 个人开发者探索 AI 与设计结合的创新应用  

**推荐理由**  
开源、本地优先、支持多 AI 工具，适合需要安全、灵活设计工具的用户，是 Claude Design 的优质替代方案。
