# aquasecurity/trivy

Find vulnerabilities, misconfigurations, secrets, SBOM in containers, Kubernetes, code repositories, clouds and more

项目定位  
Trivy 是一个开源的软件供应链安全工具，用于检测容器、Kubernetes、代码仓库、云资源中的漏洞、配置错误、敏感信息和 SBOM。

核心功能  
- 检测容器镜像中的漏洞  
- 扫描 Kubernetes 集群配置  
- 识别代码仓库中的敏感信息  
- 生成软件物料清单（SBOM）  
- 支持多种操作系统和包管理器

技术特点  
Trivy 使用 Go 语言开发，具备高性能和跨平台特性，支持快速扫描并集成到 CI/CD 流程中，提供详细的报告和修复建议。

适用场景  
- 容器化应用的安全检查  
- Kubernetes 集群的配置审计  
- 代码仓库的敏感信息扫描  
- 软件供应链安全合规  
- 云资源的安全评估

推荐理由  
Trivy 是一款功能全面、易于集成的开源安全工具，适合企业快速发现和修复软件供应链中的安全问题，提升整体安全性。
