# aquasecurity/trivy

Find vulnerabilities, misconfigurations, secrets, SBOM in containers, Kubernetes, code repositories, clouds and more

```markdown
1. 项目定位 - Trivy 是一个开源安全工具，用于发现容器、Kubernetes、代码仓库、云资源中的漏洞、配置错误、敏感信息和 SBOM。

2. 核心功能 - 检测容器镜像中的漏洞；扫描 Kubernetes 集群配置；查找代码仓库中的敏感信息；生成软件物料清单（SBOM）；支持多种云平台。

3. 技术特点 - 使用 Go 语言编写，性能高效；支持多平台扫描；集成简单，可嵌入 CI/CD 流程；拥有活跃社区和持续更新。

4. 适用场景 - 容器安全扫描；DevOps 安全检查；云原生应用安全评估；代码仓库安全审计；企业级安全合规。

5. 推荐理由 - 功能全面，易于使用，适用于现代开发和运维流程，是保障容器化和云原生环境安全的重要工具。
```
