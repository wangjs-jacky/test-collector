# chenhg5/cc-connect

Bridge local AI coding agents (Claude Code, Cursor, Gemini CLI, Codex) to messaging platforms (Feishu/Lark, DingTalk, Slack, Telegram, Discord, LINE, WeChat Work). Chat with your AI dev assistant from anywhere — no public IP required for most platforms.

项目定位  
将本地 AI 编码助手连接到多种消息平台，实现跨平台编程协作，无需公网 IP。

核心功能  
- 支持 Claude Code、Cursor 等 AI 编码工具接入  
- 连接 Feishu、Slack、Telegram 等多个消息平台  
- 实现本地 AI 与消息平台的双向通信  
- 提供统一接口，简化多平台集成流程  

技术特点  
使用 Go 语言开发，具备高性能和良好的跨平台支持，采用代理机制实现本地服务与远程平台的安全连接。

适用场景  
- 开发者在公司内网中使用本地 AI 工具  
- 团队通过消息平台协作编程  
- 需要避免暴露本地服务公网 IP 的场景  
- 快速搭建 AI 编程助手与消息平台的桥梁  

推荐理由  
功能实用，解决了本地 AI 工具与主流消息平台连接的难题，适合需要安全、便捷集成的开发者群体。
