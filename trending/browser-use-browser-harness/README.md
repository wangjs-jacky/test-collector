# browser-use/browser-harness

Browser Harness | Self-healing harness that enables LLMs to complete any task.

**项目定位**  
Browser Harness 是一个用于让大语言模型（LLM）完成任意任务的自我修复框架，通过浏览器自动化实现任务执行与问题处理。

**核心功能**  
- 通过浏览器自动化执行用户指令  
- 自我修复机制处理任务失败情况  
- 支持多种浏览器和操作系统  
- 提供任务日志与调试信息  
- 集成 LLM 作为任务执行的核心逻辑

**技术特点**  
使用 Python 实现跨平台浏览器控制，结合 LLM 的推理能力，具备任务恢复和错误处理机制，适合复杂交互场景。

**适用场景**  
- 自动化网页操作任务  
- 测试 LLM 在真实浏览器环境中的表现  
- 执行需要图形界面的脚本任务  
- 构建智能客服或自动化助手  
- 研究 LLM 与浏览器交互的可行性

**推荐理由**  
Browser Harness 提供了 LLM 与浏览器交互的完整解决方案，具备自我修复和跨平台能力，适合开发者探索 AI 在真实环境中的应用。
