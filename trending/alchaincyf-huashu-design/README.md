# alchaincyf/huashu-design

Huashu Design · HTML-native design skill for Claude Code · Claude Code 里 HTML 原生的设计 skill · 高保真原型 / 幻灯片 / 动画 + 20 设计哲学 + 5 维评审 + MP4 导出 · Agent-agnostic

项目定位  
Huashu Design 是一个为 Claude Code 设计的 HTML 原生设计技能工具，用于创建高保真原型、幻灯片和动画，支持多种设计哲学与评审维度。

核心功能  
- 高保真原型设计  
- 幻灯片与动画制作  
- 20 条设计哲学指导  
- 5 维评审体系  
- MP4 视频导出功能  

技术特点  
基于 HTML 实现原生设计能力，无需依赖特定 AI 模型，具备良好的兼容性与可扩展性，适合各类设计流程集成。

适用场景  
- AI 辅助设计流程中使用  
- 快速构建网页原型  
- 制作演示文稿与动画  
- 设计团队协作与评审  
- 无代码或低代码环境下的设计实践  

推荐理由  
Huashu Design 提供了强大的 HTML 原生设计能力，结合多维评审与动画导出，是 AI 编程环境中高效设计的有力工具。
