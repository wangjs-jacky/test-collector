# JuliusBrussee/caveman

🪨 why use many token when few token do trick — Claude Code skill that cuts 65% of tokens by talking like caveman

**项目定位**  
该项目通过简化语言风格（“原始人风格”）减少 AI 模型的 token 使用量，提升效率，适用于需要优化提示工程的场景。

**核心功能**  
- 将复杂提示转换为简单、直接的表达方式  
- 降低 AI 生成内容所需的 token 数量  
- 提供可复用的提示模板库  
- 支持与 Claude Code 等模型结合使用  
- 优化 AI 的响应质量与速度  

**技术特点**  
基于 Python 实现，利用简洁的语言结构和提示工程技巧，有效减少 token 消耗，同时保持模型输出的准确性与实用性。

**适用场景**  
- 优化 AI 提示以降低计算成本  
- 在资源受限环境下提升 AI 效率  
- 用于 AI 工具开发中的提示优化  
- 适合对 AI 输出质量有要求但希望减少 token 使用的用户  
- 适用于需要快速迭代和测试不同提示策略的开发者  

**推荐理由**  
通过创新的“原始人风格”提示方法，显著减少 token 使用，提高 AI 响应效率，是提示工程和 AI 优化领域的实用工具。
