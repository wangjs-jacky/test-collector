# emdash-cms/emdash

EmDash is a full-stack TypeScript CMS based on Astro; the spiritual successor to WordPress

项目定位  
EmDash 是一个基于 Astro 的全栈 TypeScript 内容管理系统，旨在成为 WordPress 的精神继承者，提供更现代、灵活的内容管理体验。

核心功能  
- 支持内容创作与管理  
- 前后端分离架构  
- 基于 TypeScript 的开发体验  
- 可扩展的插件系统  
- 与 Astro 深度集成，优化静态站点生成

技术特点  
EmDash 采用 TypeScript 构建，结合 Astro 实现高性能静态站点生成，具备现代化的开发工具链和模块化架构，适合构建快速、可维护的内容驱动网站。

适用场景  
- 企业官网与博客  
- 个人或团队内容平台  
- 需要快速部署的静态站点  
- 需要高度定制化的 CMS 系统  
- 与现代前端框架集成的项目

推荐理由  
EmDash 提供了现代的开发体验和强大的内容管理能力，适合希望摆脱传统 CMS 瓶颈并利用最新技术栈的开发者。
