# ronitsingh10/FineTune

FineTune, a macOS menu bar app for per-app volume control, multi-device output, audio routing, and 10-band EQ. Free and open-source alternative to SoundSource.

**项目定位**  
FineTune 是一款 macOS 菜单栏应用，提供针对不同应用程序的音量控制、多设备输出、音频路由和 10 频段均衡器功能，是 SoundSource 的开源替代方案。

**核心功能**  
- 每个应用程序独立音量控制  
- 支持多设备音频输出  
- 音频路由管理  
- 10 频段均衡器调节  
- 菜单栏快速访问

**技术特点**  
使用 Swift 编写，结合 SwiftUI 构建用户界面，支持 macOS 系统级音频控制，具备良好的性能与用户体验。

**适用场景**  
- 多任务处理时调整不同应用音量  
- 使用多个音频设备的用户  
- 需要自定义音频输出的开发者  
- 喜欢简洁菜单栏工具的用户  
- 音频爱好者进行音质调校

**推荐理由**  
FineTune 提供强大而直观的音频控制功能，开源且免费，适合需要精细管理音频输出的 macOS 用户，是 SoundSource 的优质替代品。
