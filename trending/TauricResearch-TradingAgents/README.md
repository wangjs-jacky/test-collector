# TauricResearch/TradingAgents

TradingAgents: Multi-Agents LLM Financial Trading Framework

**项目定位**  
TradingAgents 是一个基于多智能体 LLM 的金融交易框架，用于构建和测试自动化交易策略。

**核心功能**  
- 支持多智能体协同交易  
- 提供回测与实时交易功能  
- 集成 LLM 以实现智能决策  
- 可扩展的交易策略模块  
- 实时市场数据接入  

**技术特点**  
使用 Python 构建，结合 LLM 和强化学习技术，支持高并发交易模拟与策略优化，具备良好的可扩展性与灵活性。

**适用场景**  
- 量化交易策略开发  
- 金融 AI 研究实验  
- 交易所算法交易测试  
- 多智能体协作交易研究  
- 金融数据可视化分析  

**推荐理由**  
适合对金融交易和 AI 结合感兴趣的开发者，提供完整的框架与工具，便于快速构建和验证交易智能体。
