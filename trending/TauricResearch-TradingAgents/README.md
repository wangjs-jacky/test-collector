# TauricResearch/TradingAgents

TradingAgents: Multi-Agents LLM Financial Trading Framework

TradingAgents: Multi-Agents LLM Financial Trading Framework 是一个基于大语言模型的多智能体金融交易框架，旨在通过 AI 技术实现自动化、智能化的金融交易策略。

- 实现多智能体协同进行金融交易  
- 集成大语言模型进行市场分析与决策  
- 支持回测与实时交易功能  
- 提供可扩展的交易策略开发接口  
- 适用于股票、加密货币等多种金融资产  

采用 Python 编写，结合大语言模型与多智能体系统，具备良好的可扩展性与灵活性，适合金融交易领域的研究与实践。

- 量化交易策略开发与测试  
- 金融市场的 AI 分析与预测  
- 多智能体协作交易实验  
- 加密货币交易模拟与实战  
- 金融科技创新与研究  

项目定位清晰，技术先进，适合对金融与 AI 结合感兴趣的开发者与研究人员，具有较高的实用价值与学习意义。
