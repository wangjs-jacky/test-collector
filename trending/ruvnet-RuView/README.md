# ruvnet/RuView

π RuView: WiFi DensePose turns commodity WiFi signals into real-time human pose estimation, vital sign monitoring, and presence detection — all without a single pixel of video.

1. 该项目通过普通 WiFi 信号实现实时人体姿态估计、生命体征监测和存在检测，无需视频。

2. - 实时人体姿态估计  
   - 生命体征监测（如呼吸、心跳）  
   - 无视频的人员存在检测  
   - 利用 WiFi 信号进行非接触式感知  

3. 使用 Rust 编写，注重性能与安全性，结合机器学习模型从 WiFi 信号中提取人体特征，实现高精度感知。

4. - 智能家居中的人员活动监测  
   - 医疗健康监测场景  
   - 安防系统中的无感检测  
   - 无人值守环境下的行为分析  

5. 项目创新性强，利用 WiFi 信号实现非接触式人体感知，具有广泛的应用前景，代码结构清晰，技术先进。
