# ruvnet/RuView

π RuView: WiFi DensePose turns commodity WiFi signals into real-time human pose estimation, vital sign monitoring, and presence detection — all without a single pixel of video.

项目定位  
RuView 是一个利用 Wi-Fi 信号实现实时人体姿态估计、生命体征监测和存在检测的项目，无需视频数据即可完成人体感知。

核心功能  
- 实时人体姿态估计  
- 生命体征监测（如呼吸、心跳）  
- 无视频的人员存在检测  
- 利用 Wi-Fi 信号进行非接触式感知  

技术特点  
基于 Rust 编写，结合机器学习与无线信号处理技术，实现高效、低延迟的人体状态分析，具有较高的实时性和准确性。

适用场景  
- 智能家居中的健康监测  
- 公共场所的安全监控  
- 医疗健康设备的非侵入式检测  
- 无人值守环境下的人员识别  

推荐理由  
RuView 创新性地利用 Wi-Fi 信号实现人体感知，突破传统视频监控的局限，适用于隐私敏感或复杂环境，具有广泛的应用前景。
