项目定位  
一款支持多 AI 开发工具的账号管理工具，帮助开发者高效管理多个 AI IDE 账号，实现多账号切换、配额监控与自动化操作。

核心功能  
- 多账号切换与管理  
- 实时配额监控与预警  
- 自动唤醒与多实例管理  
- 支持多种 AI 工具（如 GitHub Copilot、Codex 等）  
- 配置灵活，易于扩展  

技术特点  
使用 Rust 编写，具备高性能和稳定性，支持跨平台运行，代码结构清晰，便于维护和二次开发。

适用场景  
- 开发者同时使用多个 AI 编程工具  
- 需要监控 API 使用配额的团队  
- 需要自动化管理多个 AI 实例的场景  
- 企业级 AI 工具统一管理  
- 提升 AI 工具使用效率与便捷性  

推荐理由  
功能全面，支持多种主流 AI 工具，提升开发效率与账号管理体验，适合需要多账号协作的开发者和团队。
