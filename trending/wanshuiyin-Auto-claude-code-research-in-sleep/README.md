# wanshuiyin/Auto-claude-code-research-in-sleep

ARIS ⚔️ (Auto-Research-In-Sleep) — Lightweight Markdown-only skills for autonomous ML research: cross-model review loops, idea discovery, and experiment automation. No framework, no lock-in — works with Claude Code, Codex, OpenClaw, or any LLM agent.

项目定位  
一个用于自动化机器学习研究的轻量级工具，支持跨模型评审、想法发现和实验自动化，无需依赖特定框架。

核心功能  
- 自动化论文评审与摘要生成  
- 研究想法自动生成与优化  
- 实验流程自动化与结果记录  
- 支持多种 LLM 代理（如 Claude Code、Codex）  
- 轻量级 Markdown 格式，无框架绑定

技术特点  
基于 Python 构建，采用 Markdown 作为主要输入输出格式，具备良好的可扩展性和兼容性，适合集成到各种研究工作流中。

适用场景  
- 机器学习研究员日常实验管理  
- 论文撰写与评审辅助  
- 自动化模型训练与评估流程  
- 多模型对比与分析  
- 研究思路探索与创新点挖掘

推荐理由  
轻量且灵活，适合希望提升研究效率、减少重复劳动的 ML 研究者，尤其适合结合 LLM 进行自动化研究。
