# santifer/career-ops

AI-powered job search system built on Claude Code. 14 skill modes, Go dashboard, PDF generation, batch processing.

项目定位  
一个基于 Claude Code 的 AI 驱动的求职系统，帮助用户自动化简历优化、面试准备和职位搜索流程。

核心功能  
- 14 种技能模式支持个性化求职  
- Go 语言构建的可视化仪表盘  
- 支持 PDF 简历生成与批量处理  
- CLI 工具实现高效操作  

技术特点  
使用 Go 语言开发，结合 Claude Code 实现 AI 功能，具备良好的性能和可扩展性，适合构建企业级应用。

适用场景  
- 个人求职者优化简历和面试准备  
- 企业招聘团队自动化筛选候选人  
- 教育机构用于职业发展辅导  
- 自动化批量处理多个职位申请  

推荐理由  
功能全面、技术成熟，适合希望提升求职效率和质量的用户，尤其适合技术背景的开发者和求职者。
