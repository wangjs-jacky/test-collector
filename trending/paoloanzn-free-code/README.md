# paoloanzn/free-code

[NOTE] Parent repo is migrating ownership, until the operation is done this repo is blocked. The free build of Claude Code. All telemetry removed, security-prompt guardrails stripped, all experimental features enabled.

项目定位  
一个去除了所有监控和安全限制的 Claude Code 免费版本，提供实验性功能和更自由的使用体验。

核心功能  
- 提供 Claude Code 的免费版本  
- 移除所有遥测和安全提示机制  
- 启用所有实验性功能  
- 适用于开发者进行代码生成和辅助编程  

技术特点  
基于 TypeScript 构建，专注于去除安全限制和监控功能，提供高度可定制的代码生成体验。

适用场景  
- 开发者快速生成代码片段  
- 需要绕过安全限制的测试环境  
- 实验性功能的探索与使用  
- 个人项目中的代码辅助生成  

推荐理由  
适合需要自由使用 Claude Code 功能的开发者，尤其在安全限制严格的环境中提供灵活的替代方案。
