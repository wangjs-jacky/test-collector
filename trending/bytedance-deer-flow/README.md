# bytedance/deer-flow

An open-source SuperAgent harness that researches, codes, and creates. With the help of sandboxes, memories, tools, skill, subagents and message gateway, it handles different levels of tasks that could take minutes to hours.

**项目定位**  
一个开源的超级代理框架，用于研究、编码和创造，通过沙盒、记忆、工具等组件处理复杂任务。

**核心功能**  
- 支持多级任务处理  
- 集成沙盒环境进行安全测试  
- 提供记忆存储与调用机制  
- 支持子代理协作  
- 通过消息网关进行通信  

**技术特点**  
基于 Python 构建，融合了沙盒、记忆、工具和子代理等技术，实现高效的任务管理和执行能力。

**适用场景**  
- 自动化复杂任务处理  
- 研究型 AI 项目开发  
- 安全测试与代码验证  
- 多代理系统构建  
- 长周期任务调度  

**推荐理由**  
功能强大且结构清晰，适合需要处理复杂任务和多代理协作的开发者，是 AI 研究和应用的优秀工具。
