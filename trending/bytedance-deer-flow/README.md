# bytedance/deer-flow

An open-source SuperAgent harness that researches, codes, and creates. With the help of sandboxes, memories, tools, skill, subagents and message gateway, it handles different levels of tasks that could take minutes to hours.

```markdown
1. 项目定位  
一个开源的 SuperAgent 框架，用于研究、编码和创造，通过沙盒、记忆、工具等模块处理复杂任务。

2. 核心功能  
- 支持多层级任务处理  
- 提供沙盒环境进行安全执行  
- 集成记忆模块保存上下文信息  
- 支持子代理和消息网关通信  

3. 技术特点  
基于 Python 实现，结合沙盒、记忆、工具和子代理等技术，构建灵活的自动化任务处理系统。

4. 适用场景  
- 自动化复杂流程任务  
- 研究型 AI 项目开发  
- 多代理协作系统构建  
- 安全执行外部代码  

5. 推荐理由  
高星项目，功能强大且结构清晰，适合需要处理复杂任务的开发者和研究人员使用。
```
