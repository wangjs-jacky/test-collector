# JackChen-me/open-multi-agent

TypeScript multi-agent framework — one runTeam() call from goal to result. Auto task decomposition, parallel execution. 3 dependencies, deploys anywhere Node.js runs.

1. **项目定位**  
一个基于 TypeScript 的多智能体框架，通过 `runTeam()` 实现从目标到结果的自动化任务分解与并行执行，适用于多种 AI 模型和工具。

2. **核心功能**  
- 自动任务分解  
- 并行任务执行  
- 支持多种 LLM 和工具集成  
- 结构化输出  
- 任务调度与编排  

3. **技术特点**  
使用 TypeScript 构建，依赖少且部署灵活，支持 Node.js 环境，具备模型无关性和良好的扩展性。

4. **适用场景**  
- 复杂任务自动化处理  
- 多 AI 模型协同工作  
- 需要结构化输出的应用  
- 需要任务调度和并行执行的系统  
- 跨平台 AI 工具集成  

5. **推荐理由**  
功能强大且易于集成，适合需要多智能体协作和自动化任务处理的开发者，具备良好的灵活性和可扩展性。
