# Donchitos/Claude-Code-Game-Studios

Turn Claude Code into a full game dev studio — 48 AI agents, 36 workflow skills, and a complete coordination system mirroring real studio hierarchy.

项目定位  
该项目旨在将 Claude Code 转化为一个完整的游戏开发工作室，通过 AI 代理和工作流技能提升游戏开发效率。

核心功能  
- 48 个 AI 代理协同工作  
- 36 种工作流技能支持开发流程  
- 模拟真实游戏工作室的层级协调系统  
- 支持多种游戏引擎如 Unity、Unreal Engine 和 Godot  
- 提供 AI 辅助开发工具链

技术特点  
项目利用 Shell 脚本构建自动化流程，结合 AI 技术实现游戏开发的智能化与协作化，适合多平台游戏开发。

适用场景  
- 游戏开发团队提高效率  
- 独立开发者快速原型制作  
- AI 技术在游戏设计中的探索  
- 教育和研究 AI 在游戏开发中的应用  
- 多引擎跨平台开发实践

推荐理由  
该项目展示了 AI 在游戏开发中的强大潜力，提供了一套完整的工具和架构，适合希望探索 AI 辅助开发的开发者。
