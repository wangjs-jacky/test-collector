# vas3k/TaxHacker

Self-hosted AI accounting app. LLM analyzer for receipts, invoices, transactions with custom prompts and categories

项目定位  
TaxHacker 是一个自托管的 AI 会计应用，用于分析收据、发票和交易，支持自定义提示和分类，帮助用户更高效地管理财务。

核心功能  
- AI 分析收据、发票和交易数据  
- 支持自定义提示和分类规则  
- 集成多种 LLM 模型（如 Gemini、OpenAI）  
- OCR 识别与货币转换功能  
- 自托管部署，保护数据隐私

技术特点  
使用 TypeScript 开发，支持多种 AI 模型集成，具备 OCR 和货币转换能力，适合需要自托管解决方案的用户。

适用场景  
- 个人财务管理  
- 小型企业税务处理  
- 多币种交易记录  
- 需要数据隐私保护的场景  
- 快速自动化账目分类

推荐理由  
TaxHacker 提供强大的 AI 分析功能，支持自定义和多模型集成，适合希望自主管理财务并提升效率的用户。
