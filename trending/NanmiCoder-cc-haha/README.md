# NanmiCoder/cc-haha

Claude Code 泄露源码 - 本地可运行版本（附带核心模块解析）

项目定位  
该项目是一个 Claude Code 泄露源码的本地运行版本，提供核心模块解析，便于开发者研究和测试。

核心功能  
- 提供 Claude Code 的本地可运行版本  
- 包含核心模块的解析与实现  
- 支持代码生成与推理功能  
- 用于研究大模型的内部结构与逻辑  

技术特点  
基于 TypeScript 实现，具备良好的可读性与扩展性，适合进行代码级分析与二次开发。

适用场景  
- 研究大模型的代码结构与实现方式  
- 本地测试与调试 Claude Code 相关功能  
- 教学与学习大模型的工作原理  
- 开发者进行代码分析与逆向工程  

推荐理由  
适合对大模型技术感兴趣的开发者和研究人员，提供了一个可运行、可分析的 Claude Code 源码环境，便于深入理解其工作原理。
