# webadderall/Recordly

Create polished screen recordings for free. An open-source screen recorder for Mac/Windows/Linux that adds auto-zooms, animated cursors, auto-captions and more to your videos.

**项目定位**  
Recordly 是一个开源的屏幕录制工具，支持多平台，提供自动缩放、动画光标和自动字幕等功能，帮助用户轻松制作专业级的屏幕录像。

**核心功能**  
- 自动缩放功能提升视频观看体验  
- 动画光标增强操作可视化  
- 自动字幕生成提高内容可访问性  
- 支持 Mac/Windows/Linux 多系统  

**技术特点**  
使用 TypeScript 开发，基于 Electron 构建跨平台应用，具备良好的性能和兼容性，代码结构清晰，适合开发者参与贡献。

**适用场景**  
- 在线教学与课程录制  
- 软件演示与产品展示  
- 游戏直播与内容创作  
- 会议记录与远程协作  

**推荐理由**  
Recordly 提供免费且功能丰富的屏幕录制体验，适合教育、开发及内容创作者，开源特性也便于定制与扩展。
