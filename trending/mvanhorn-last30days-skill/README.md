# mvanhorn/last30days-skill

AI agent skill that researches any topic across Reddit, X, YouTube, HN, Polymarket, and the web - then synthesizes a grounded summary

```markdown
1. 项目定位：一个 AI 助手技能，能够跨平台研究任意主题并生成基于事实的总结，帮助用户快速获取信息。

2. 核心功能：  
   - 跨 Reddit、X、YouTube、HN、Polymarket 和网络搜索信息  
   - 自动整合多来源数据  
   - 生成结构化、可验证的摘要  
   - 支持对复杂话题进行深入分析  

3. 技术特点：基于 Python 构建，利用 API 和爬虫技术实现多平台数据采集与处理，结合自然语言处理技术生成高质量摘要。

4. 适用场景：  
   - 快速了解热点事件或趋势  
   - 分析市场预测或投资机会  
   - 汇总学术或行业研究资料  
   - 辅助内容创作与决策支持  

5. 推荐理由：功能强大且实用，能高效整合多源信息，适合需要快速获取全面信息的用户，尤其在信息过载的环境下具有显著优势。
```
