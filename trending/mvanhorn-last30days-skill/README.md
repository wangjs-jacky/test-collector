# mvanhorn/last30days-skill

AI agent skill that researches any topic across Reddit, X, YouTube, HN, Polymarket, and the web - then synthesizes a grounded summary

**项目定位**  
该项目是一个 AI 代理技能，用于跨平台研究任意主题，整合 Reddit、X、YouTube、HN、Polymarket 和网络信息，生成有依据的摘要。

**核心功能**  
- 跨平台数据采集（Reddit、X、YouTube 等）  
- 自然语言处理与内容理解  
- 信息综合与摘要生成  
- 实时数据更新与动态分析  

**技术特点**  
基于 Python 构建，结合 API 调用与 NLP 技术，实现多源信息整合与智能摘要，具备良好的扩展性和实时性。

**适用场景**  
- 新闻热点追踪  
- 市场趋势分析  
- 社交媒体舆情监控  
- 学术或商业研究支持  
- 个人知识管理  

**推荐理由**  
适合需要快速获取多平台信息并生成结构化摘要的用户，尤其适用于研究、分析和决策支持场景。
